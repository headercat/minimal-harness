# minimal-harness

Minimal, safe tool harness for LLM agents.

## Installation

```bash
npm install @minimal-harness/core
```

## Packages

This monorepo contains the following packages:

- `@minimal-harness/core` — Core harness for defining and running LLM-powered tools
- `@minimal-harness/ai-sdk` — Integration with Vercel AI SDK
- `@minimal-harness/telegram` — Telegram bot adapter
- `@minimal-harness/tools` — Pre-built utility tools
